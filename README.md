# Movie Ratings & Reviews App

This is a simple movie ratings and reviews app built with HTML, CSS, and Node.js.

## File Structure

```
project-root/
│
├── views/
│   ├── index.ejs
│   ├── movie-detail.ejs
│   └── submit-review.ejs
├── public/
│   ├── styles.css
│   └── placeholder.jpg
├── node_modules/
├── app.js
├── package.json
└── package-lock.json
```

## How to Run the App

1. **Clone the repository:**
   ```sh
   git clone <repository-url>
   cd project-root
   ```

2. **Install the dependencies:**
   ```sh
   npm install
   ```

3. **Run the application:**
   ```sh
   node app.js
   ```

4. **Open your web browser and go to:**
   ```
   http://localhost:3000
   ```
